# Todo List
A simple Task Tracker used to track and manage tasks.

## Core Features
- A command-line interface (CLI)
  - GUI possible in the future
- Task tracking
  - Add tasks
  - Eidt tasks
  - Completed tasks
  - Displaying all tasks
  - Tracking of tasks you're working on
- Task Properties
  - ID: A unique identifier for each task.
  - Description: A short description of the task.
  - Status: The status of the task ('todo', 'in-progress', 'done').
  - Date_Created: The day and time the task was made.
  - Date_Updated: The day and time the task was updated.
